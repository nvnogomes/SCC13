Upper set

From Wikipedia, the free encyclopedia

     (Redirected from [6]Down set lattice)
   Jump to: [7]navigation, [8]search

   In [9]mathematics, an upper set, or upward [10]closed set, is a subset
   Y of a given [11]partially ordered set (X,) such that, for all elements
   x and y, if x is less than or equal to y and x is an element of Y, then
   y is also in Y. More formally,

          \forall x\forall y\left[x \le y \and x \in Y \Rightarrow \; y
          \in Y\right].

   The [12]dual notion is lower set (alternatively, down set, decreasing
   set, initial segment; the set is downward closed), which is any subset
   Y of a given partially ordered set (X,) such that, for all elements x
   and y, if x is less than or equal to y and y is an element of Y, then x
   is also in Y. More formally,

          \forall x\forall y\left[x \le y \and y \in Y \Rightarrow \; x
          \in Y\right].

[[17]edit] Properties

   Every partially ordered set is an upper set of itself. The
   [18]intersection of upper sets is again an upper set. The
   [19]complement of any upper set is a lower set, and vice versa.

   Given a partially ordered set (X,), the family of lower sets of X
   ordered with the [20]inclusion relation is a [21]complete lattice, the
   down-set lattice O(X).

   Given an arbitrary subset Y of an ordered set X, the smallest upper set
   containing Y is denoted using an up arrow as Y. Dually, the smallest
   lower set containing Y is denoted using a down arrow as Y. A lower set
   is called principal if it is of the form {x} where x is an element of
   X. Every lower set Y of a finite ordered set X is equal to the smallest
   lower set containing all [22]maximal elements of Y: Y = Max(Y) where
   Max(Y) denotes the set containing the maximal elements of Y.

   A [23]directed lower set is called an [24]order ideal.

   The [25]minimal elements of any upper set form an [26]antichain.
   Conversely any antichain A determines an upper set {x: for some y in A,
   x y}. For partial orders satisfying the [27]descending chain condition
   this correspondence between antichains and upper sets is 1-1, but for
   more general partial orders this is not true.

[[28]edit] Ordinal numbers

   An [29]ordinal number is usually identified with the set of all smaller
   ordinal numbers. Thus the ordinal numbers are exactly the lower sets of
   ordinal numbers.
