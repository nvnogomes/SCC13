XNOR gate

   The XNOR gate (sometimes spelled 'exnor') is a digital [9]logic gate
   whose function is the inverse of the exclusive OR ([10]XOR) gate. The
   two-input version implements [11]logical equality, behaving according
   to the truth table to the right. A HIGH output (1) results if both of
   the inputs to the gate are the same. If one but not both inputs are
   HIGH (1), a LOW output (0) results.

[[16]edit] Symbols

   There are two symbols for XNOR gates: the 'professional' symbol and the
   'rectangular' symbol. For more information see [17]Logic Gate Symbols
   'Military' XNOR Symbol
   'Rectangular' XNOR Symbol

   A XNOR B= A.B+/A./B

[[18]edit] Hardware description and pinout

   XNOR Gates are basic logic gates, and as such they are recognised in
   [19]TTL and [20]CMOS [21]ICs. The standard, [22]4000 series, CMOS IC is
   the 4077, which includes four independent, two-input, XNOR gates. The
   pinout diagram is as follows:
   Pinout diagram of a 4077 Quad XNOR [23]DIP-format IC
 1  Input A1
 2  Input B1
 3  Output Q1
 4  Output Q2
 5  Input B2
 6  Input A2
 7  V[ss]
 8  Input A3
 9  Input B3
 10 Output Q3
 11 Output Q4
 12 Input B4
 13 Input A4
 14 V[dd]

   This device is available from most semiconductor manufacturers such as
   Philips. It is usually available in both through-hole [24]DIP and
   [25]SOIC format. Datasheets are readily available in most [26]datasheet
   databases.
