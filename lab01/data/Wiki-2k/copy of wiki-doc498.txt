Dumpfile

From Wikipedia, the free encyclopedia

   Jump to: [6]navigation, [7]search

   A dumpfile is an [8]output file used to save detailed step-by-step
   results of a [9]simulation run for analysis, re-simulation, or design
   verification.

   Dump files can be accessed through the task manager on a Microsoft
   Windows OS (Operating System)
